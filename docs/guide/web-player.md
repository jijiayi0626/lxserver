# Web 播放器使用向导

除了管理同步服务器，本系统还在 `/music` （即 `public/music/index.html`）提供了一个功能强大的现代化 Web 音乐播放器。这代表你不仅仅可以管理云端数据，你随时随地打开网页，就能聆听被同步上来的音乐库。

## 如何访问

访问您的服务地址路径 `/music`：
**示例**：`http://your-server-ip:9527/music`

如果开启了安全认证，在弹出的窗口输入密码（管理员可配置开启：`ENABLE_WEBPLAYER_AUTH` 环境变量）。

## 核心功能简介

### 1. 与服务端歌单联动

Web 播放器自动拉取您在 LX Music 其他客户端（如电脑端或手机端 APP）所存储的网名与歌单，保证数据与同步服务器的数据完全对称，无缝衔接。

### 2. 多源音乐聚合搜索

Web 播放器内部通过特殊的源（Source）设计，使得你能用统一的方式搜索主流音乐平台（QQ、网易、KuGou、KuWo、Migu等）的歌曲：

- **智能搜索**：不局限于本地，可以像其他音乐播放器一样检索在线内容。
- **自定义音源脚本**：你可以将自定义的脚本通过设置面板导入至系统内部增强抓取成功率。

### 3. 先进的音频播放控制功能

- **倍速播放控制**
- **音质选择**
- **歌词支持（包括动效显示）**
- **睡眠定时设置功能**

### 4. 样式与主题支持

为了最佳视听享受，Web 播放支持切换包括**各种主题色**等外观。并且可以在手机移动端获取与原生 App 近似的操作体感。在极小的窄屏手机或宽大的显示器上都有不一样的弹性自适应设计。

---

> ⚠️ 与管理台功能不互通
> Web 播放器仅仅只用来播放与搜索音乐。如果你需要清理某个失效的连接、配置 WebDAV 计划，请返回访问控制台 `http://your-server-ip:9527` 查阅《同步服务器管理指南》。
