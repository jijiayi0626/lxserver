# 同步服务器管理指南

LX Music Sync Server 致力于成为您的音乐资产管理中心。管理控制台页面（即 `public/index.html`，通过访问根路径 `/` 即可进入）提供了关于同步、设备及配置等所有后端数据的可视化管理功能。

## 登录系统

首次进入控制台界面（默认地址 `http://IP:9527`），您会被要求输入认证密码。

- **默认密码**：`123456`
- **更改密码**：可通过环境变量 `FRONTEND_PASSWORD` 进行初始覆盖，或登录后在配置界面修改。

## 核心功能详解

同步服务管理控制台主要包含以下几大功能模块，可以在侧边栏快速切换：

### 1. 仪表盘 (Dashboard)

展示目前运行的 LX Music 同步服务的**总体状态**与统计。您可以在此快速了解：
- **运行概况**：在线设备数量、系统内存占用、持续运行时间（Uptime）。
- **最近动态**：监控近期是否存在异常报错或断连事件。

### 2. 用户与设备管理

作为一个可供多人/多端连入的私有云节点。此面板允许进行接入授权级别的安全管理。

- **创建新用户**：为您的家人朋友分配相互隔离的独立歌单密码账户。
- **设备审查**：当多用户或者多端设备连接上了本同步服务（如桌面端 LX Music Client、移动端），您可以直观看到接入状态。
- **数据清理**：彻底删除某个账号时，可一并选择清理其云端遗留数据。

### 3. 数据查看 (Data Viewer)

这是帮助您排查“为什么某个歌单没同步过来”的终极利器。

- **歌单级阅览**：可视化直观地查看当前全量同步的**音乐列表**与**收藏歌曲数据**，分类进行检视。
- **快照管理与回滚**：当您的客户端发生误删等灾难时，可于此处选择系统中保留的 **历史快照数据** 下载或者执行恢复。

### 4. WebDAV 备份配置

在这里，您可以灵活配置 **WebDAV** 后端映射。这是本服务极其重要的增强特性，它会将您当前的同步数据**自动定期备份**至其他云盘（如阿里云盘 WebDAV、坚果云 Nutstore 等）。

- **原理启示**：只要配置了合法的 WebDAV 信息，服务端即使处于新服务器或新容器刚启动状态，也会第一时间自动去该 WebDAV 网盘将最新备份完整下载至本地并恢复业务，从而起到跨地域灾备迁移的作用。
- 支持开启关闭及设置备份频率（建议：`60` 分钟）。

### 5. 文件与系统日志

您可以即时查看并清理服务端的运行日志，用于系统出错时的状态排查。

---

> 💡 **提示**: 我们一直推崇将所有的设置尽量放入 Web 页面中修改，这使得不需要操作复杂的服务器后台命令行即可完全管控本应用。
